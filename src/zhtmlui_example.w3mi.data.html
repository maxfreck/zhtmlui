HTML user interface in action

About

The ZHTMLUI framework helps developers create an HTML-based user interface in classic SAP transactions.

Getting started

The frontend source code (html, css, javascript) must be stored as Binary data for WebRFC applications (transaction SMW0).

ABAP code

Because of the nature of object constructors in ABAP programming language, you have to call 2 methods for initialization:

data(html_control) = new zcl_html_ui(
  parent = cl_gui_custom_container=>screen0 "Parent GUI container
  template = 'ZHELLOWORLD'                  "Index template name from SMW0 Binary data
).
html_control->construct( ). "Actual object constructor

Yes, it's a bit ugly, but it's ABAP.

HTML setup

You have to make sure you link the files properly in your page.

<!DOCTYPE html>
<html>
  <head>
    <!-- Force IE to use latest standards -->
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta charset="utf-8">
  </head>

  <body>

  <!-- JavaScript at end of body -->
  <script src="sapguilogger.js"></script>
  <script src="sapgui.js"></script>
  </body>
</html>

Now you can enjoy the benefits of the javascript console and interact with the backend ABAP code.
